# Company app 

This project easily allows us to have control over our company(employees,tasks,departments,etc)



# Description

The application lets us to add , check , remove employees or departments from our MySQL databases . Every file (company, department,task) has business logic (bl) and db (databases).
In our db(databases) files we basically have requests for our databases. In business logic files we create logic , methods which alow us to add , remove or in other words control
our company in main class , calling already created methods . In the python console we type "help" and we are able to see all methods that we can use . After choosing a method we 
will be provided with the required information . If we miss an information or give a number where we have to give a name , the program will raise an error.Project skeleton.jpg
show the connection between every class . Database.jpg show the connection between databases connected by id .

# Technology

Python 3.7

# How can we run it

First, download the source code . Go to the Windows command prompt. One option is to choose Run from the Windows Start menu, type cmd, and click OK.
Use the "cd" command to change to the folder containing the program you wish to run. Run the command line program by typing its name and pressing 'Enter'.
